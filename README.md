# CV Application

A responsive résumé builder created with React for The Odin Project. Users can edit general information, add or remove education and experience entries, manage work responsibilities, and preview a clean version of the finished résumé.

## Live Demo

[View the live app](https://cv-application-bsul24.netlify.app/)

## Features

- Edit personal contact information
- Add, edit, and delete education entries
- Add, edit, and delete work experience entries
- Dynamically add or remove job responsibilities
- Preview the résumé without editing controls
- Responsive layout for desktop and mobile

## Built With

- React
- JavaScript
- CSS
- Vite
